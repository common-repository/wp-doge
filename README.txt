== Description ==

* Shows WP post categories or tags in a “nice”, “tasteful”, doge-speak box anywhere the [wpdoge] shortcode is placed.

* That’s pretty much it.

== Installation ==

* Install and activate. Use the [wpdoge] shortcode anywhere you'd like to display the dogespeak box.

* Use the post categories: [wpdoge use=“cats”]
* Use the post tags: [wpdoge use=“tags”]

== Frequently Asked Questions ==

* The CSS is a bit off on my site. What do I do?
